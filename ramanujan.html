Ramanujan Programming Contest

Can you compute the 10001st prime number?
Or Find the sum of digits of 2100?

A programming contest inspired by Ramanujan and his love for numbers and equations. The contest will be based on mathematical puzzles, famous equations, recurrence relations, number games etc. No perquisite knowledge in pure mathematics is required. Questions will be simple, logical and anyone who has fascination for puzzles can participate. Knowledge of any one programming language is required.

Rules and Regulations

  1. C/C++, Java are the only programming languages to be used for programming.
  2. Only one round will take place.
  3. If answers of the two persons clashes then the preference will be given to those who have shorter code, good algorithm and less executable time.
  4. A set of six mathematical/computer programming challenges will be provided to you at the start of the competition.
  5. Time for solving will be decided on the venue.
  6. Solution to all those challenges need to be submitted before the time is up.
  7. The correct answer should be displayed by running the program.
  8. Multiple use of language is not allowed.
  9. Answers corrected to two places of decimal shall only be accepted.

Tutorial

The Fibonacci sequence is defined by the recurrence relation:

Fn = Fn-1 + Fn-2, where F1 = 1 and F2 = 1. Hence the first 12 terms will be:
F1 = 1
F2 = 1
F3 = 2
F4 = 3
F5 = 5
F6 = 8
F7 = 13
F8 = 21
F9 = 34
F10 = 55
F11 = 89
F12 = 144

The 12th term, F12, is the first term to contain three digits.
What is the first term in the Fibonacci sequence to contain 1000 digits?

Solution

//code to find the sum of the digits of 2 raised to the power of 1000

#include
#include

using namespace std;

int main()
{
    vector power;
    power.push_back(1);
    int n=0,x=0;
    int a=0;
    for(int j=1;j<1000;j++)
    {
	// the vector power stores the digits of 2^n which are used one by one to calculate 2^(n+1)
        for(int i=0;i<power.size();i++)
        {
            n=(power[i]*2)+x;	// n finds the double of a digit added to some left over after
            doubling previous digit if any
            x=0;
            if(n<10)
            {
            	// u use the same position because you cannot leave the digits
                // of previous numbers stored in the array,u have to somehow remove them
                power[i]=n;
            }
            
            // u have to separate the digits if a double digit number is to be stored
            if(n>=10)
            {
                power[i]=n%10;
                x=n/10;
            }
        }
        
        //at times like 64, here 64*2=128 u have to create a new position
        if(x>0)
        {
        	power.push_back(x);
        }
        x=0;
    }
    for(int i=0;i<power.size();i++)
    	a=a+power[i];	//add all the digits
    a--;
    cout<<"\n"<<a;
    cin>>n;
}
